# camostatIPDMA
This .R file contains analysis code from the individual patient data meta-analysis "Individual Patient Data Meta-Analysis and Systematic Review Evaluating Camostat Mesilate to Treat COVID-19". Justin Lee, Yiwen Wang, and Haley Hedlin created this code. 

Note: We are unable to share all data used in the analysis, so the code file will not run. This code is being shared to provide readers with the details of the analyses conducted in the paper.
